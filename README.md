# Como executar o projeto

## Front end web
Pré-requisitos: npm || yarn

```bash
# clonar repositório
git clone https://github.com/jamillyp/fazai.git

# instalar dependências
yarn install || yarn
ou
npm install || npm i

# executar o projeto
yarn start
ou
npm start
```
